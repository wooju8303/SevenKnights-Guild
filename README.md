# 📖 공부 길드전 📖

**세븐나이츠 공부 길드** 전용 길드전 관리 웹페이지입니다.  
방어팀별 공격 전략 공유, 추천 방어팀 관리, PVE 공략을 제공합니다.

## 🚀 주요 기능

### ⚔️ 길드전 공격하기

- 상대 방어팀 선택 → 공격팀 조합 확인
- 속공, 반지, 펫, 진형, 스킬순서, 장비세팅 정보 제공
- 공격덱 제보 기능 (일반 유저)

### 🛡️ 추천 방어팀 보기

- 추천 방어팀 목록 및 상세 정보
- 방어덱 제보 기능 (일반 유저)

### 🏰 PVE 공략실

- 공성전 / 강림원정대 공략
- 턴/라운드별 공략 관리
- 공략 복사/붙여넣기(공유) 기능

### 🔐 관리자 기능

- 방어팀 / 공격팀 추가·수정·삭제 (확인 팝업 포함)
- 제보 승인·거절 시스템
- PVE 공략 편집

## 🔑 로그인 방식

- **입장 코드** 입력으로 접속 (3시간 유효)
- 코드 인증은 Firebase `allowedCodes` 경로 기반
- 관리자는 별도 비밀번호로 접속

## 🛠 기술 스택

- HTML / CSS / JavaScript (단일 파일)
- Firebase Realtime Database (데이터 저장)
- Study Library 테마 (네이비 + 앰버/골드 UI)
- 반응형 디자인 (PC/모바일 최적화)
