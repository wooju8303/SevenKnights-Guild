# 🔥 공부 길드전 🔥

**세븐나이츠 공부 길드** 전용 길드전 관리 웹페이지입니다.  
길드원 출석 체크, 길드전 점수 기록 및 관리 기능을 제공합니다.

## 🚀 주요 기능

- 길드원 로그인 (닉네임 입력)
- 길드전 점수 입력 및 기록
- 출석 체크
- 길드원 관리 (추가/삭제)
- 반응형 디자인 (PC/모바일 최적화)

## 🛠 기술 스택

- HTML / CSS / JavaScript
- Burning Soul 테마 (다크 레드 UI)
